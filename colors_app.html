Colors Web Application

The Colors Web Application is a lightweight tool that allows users to store and manage their favorite colors. Users can add new colors to a MySQL database and later search for them efficiently. The project highlights database integration and a clean, responsive interface.

  • Built with HTML, CSS, and JavaScript
  • Integrated with a MySQL database
  • Supports adding new colors to the database
  • Provides search functionality to retrieve stored colors
  • Responsive design for desktop and mobile

Tech Stack: HTML, CSS, JavaScript, MySQL

Skills Learned

  • Applied front-end development with HTML, CSS, and JavaScript
  • Integrated a MySQL database with a web application
  • Designed and implemented CRUD (Create, Read, Update, Delete) operations
  • Improved knowledge of responsive and accessible UI design
  • Gained experience connecting client-side code to a backend database

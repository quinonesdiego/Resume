Virtual Machine Project

My Virtual Machine (VM) project simulates the execution of a stack-based machine similar to the P-Machine used in compiler design. The VM was written in C and supports instructions for arithmetic operations, function calls, memory management, and program control flow.

  • Developed in C for Unix systems
  • Implements registers, stack, and instruction execution cycle
  • Supports function calls, nested scopes, and recursion
  • Outputs cycle-by-cycle execution for debugging

Tech Stack: C, Unix, Git
